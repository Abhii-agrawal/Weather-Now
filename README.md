# 🌤️ Weather-Now

Weather-Now is a simple and responsive weather web application that provides real-time weather updates for any city. It displays key weather details like temperature, humidity, and conditions in a clean UI.

---

## 🚀 Features

* 🌍 Search weather by city name
* 🌡️ Real-time temperature display
* 💧 Humidity and weather conditions
* ⚡ Fast and lightweight
* 📱 Responsive design (works on mobile & desktop)

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* Weather API (like OpenWeatherMap)

## ⚙️ Setup Instructions

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/Weather-Now.git
   ```

2. Open the project folder:

   ```bash
   cd Weather-Now
   ```

3. Open `index.html` in your browser

---

## 🔑 API Setup (Important)

To fetch real-time weather data, you need an API key:

1. Go to OpenWeatherMap
2. Sign up and generate an API key
3. Replace the API key in `script.js`

Example:

```javascript
const apiKey = "YOUR_API_KEY";
```
